# Grant Shelf Source Plan

Current live collector:
- Grants.gov public search API

Search areas:
- Climate / environment / resilience
- Youth / education / skills
- Digital / data / technology / civic innovation
- Health / mental health / community wellbeing
- Culture / archives / heritage
- Caribbean / Trinidad / Tobago / SIDS / islands

Important limitation:
- Grants.gov contains real public opportunities, but many are U.S.-focused.
- Grant Shelf treats records as "Live Source" but still flags eligibility verification.
